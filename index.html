Enhanced Retirement Simulator

Build and test retirement plans using Monte Carlo modelling, Guyton-Klinger guardrails, two-person state pension offsets, cashlike buffers, and deterministic base-case projections.

Inputs

All planning and modelling controls on one page so more functionality stays above the fold.

Core assumptions

Retirement horizon, portfolio size, starting spend, and allocation mix.

Edit either the starting withdrawal rate or the starting spend. The other field updates automatically.

State pension

Current ages, pension ages, and today’s annual pension amounts for both people.

Withdrawal policy

Guyton-Klinger guardrails plus the inflation-skip rule used by the simulator.

Market assumptions

Expected return, volatility, and inflation assumptions used for deterministic and Monte Carlo paths.

Simulation settings

Monte Carlo controls.

The simulator can keep a high portfolio survival rate even when starting spend is aggressive, because the guardrails are allowed to cut spending over time. Use the yearly results and spending chart to see how far withdrawals are being reduced.

Results

Charts and yearly outputs from the current deterministic base case and Monte Carlo engine, including the effect of Guyton-Klinger guardrails.

Chart mode
Guardrails

Nominal shows the future pound amounts you would actually see at the time.

Real removes inflation so everything is shown in today’s spending power.

For retirement income planning, real view is usually the better guide because the guardrails adjust spending using inflation.

Portfolio projection

Shows the central outcome, the likely range of outcomes, and the base-case path.

Spending path

Shows target spending, actual spending after guardrail changes, pension income, and portfolio withdrawals.

Yearly results

Shows the year-by-year base-case path including spending, pension income, withdrawals and portfolio value.
